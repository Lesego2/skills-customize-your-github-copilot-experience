
# 🎮 Hangman Game Challenge

Build the classic word-guessing game using Python strings, loops, and user input.

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts.

**Skills practiced:** String manipulation, loops, conditionals, random selection

## ✅ Tasks

### 🎮 1. Set up word list
#### Description
Create a list of words for the game to randomly choose from.

#### Requirements
- Store at least 10 words in a Python list
- Randomly select one word each game using random.choice()

### 🎮 2. Handle user guesses
#### Description
Let the player guess letters and track progress.

#### Requirements
- Accept letter input from user
- Show current progress with _ for unguessed letters
- Track incorrect guesses and remaining attempts
- End game when word is guessed or attempts run out
